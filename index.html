        Going from 2 to 3 vertices increased the number of vertices by 1, but increased the
        number of edges by 2.
      

      

        When you add a vertex (a dot), the number of edges (lines) also
        increases. But — except when going from 0 vertices to 1 or from
        one vertex to two — the increase in edges is always greater than
        the increase in vertices.
