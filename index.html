Now, let's talk about what a "triangular number" is.

You get a triangular number for an input number, say, n, by doing the following:

  1. Draw an equilateral triangle with n dots on each side.
  2. Count the total number of dots in the triangle.

The number of dots in the triangle is the triangular number of n.
