          A graph is a collection of
          vertices and edges. Here are two examples:
        


        

        

          There is no requirement that vertices must be connected by edges,
          though, they can be connected by edges. There isn't even a
          requirement that there are any edges at all. A single vertex can be a
          graph.
        


        
      

    


    

      

        A complete graph is a graph in which
        every vertex is connected to every other vertex in the graph by an edge.
      


      

        This is a complete graph for
        3 vertices:
      


      

      
      

        Move the slider to change in the number of vertices (which in turn will
        really change the number of edges).
      

      

        Generally, when you increase the number of vertices in a complete graph
        by 1, the number of edges increases by v + e, where v is the
        number of vertices you had before the increase, and e is the number
        edges you had before the increase.
      

      

        For example, increasing v, the number of vertices from
        2 to
        3 vertices
        increases e, the number of edges, from
        1 to
        3 (v = 2, e
        = 1, 2 + 1 = 3). So, v increased by 1, but e increased by 2.
