        A complete graph is a graph in which every vertex is connected to every
        other vertex in the graph by an edge.

        Move the slider to change in the number of vertices (which in turn will really change the number of edges).
      

      
Note that, generally, when you increase the number of vertices in a complete graph, the number of edges increases even more.
        For example, going from 2 to 3 vertices increases the number of vertices in the graph
        by 1, but increases the number of edges by
        2.
